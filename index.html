3
Enable Google Sheets API
APIs & Services → Library → search "Google Sheets API" → Enable.
4
Create OAuth Client ID
Credentials → Create Credentials → OAuth Client ID → Web Application. Under Authorised JavaScript origins add your GitHub Pages URL.
5
OAuth Consent Screen
Add scopes: spreadsheets + drive.file. Add your email as test user. Publishing = anyone can sign in.
6
