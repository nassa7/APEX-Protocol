Google Sheets Setup — 5 Min Guide
You need a Google Cloud OAuth Client ID. This is free and takes 5 minutes. Follow these steps exactly.
1
Go to Google Cloud Console
Open this link on your laptop (easier than phone):
console.cloud.google.com
2
Create a new project
Click the project dropdown at the top → "New Project" → name it APEX Protocol → Create.
3
Enable Google Sheets API
Go to APIs & Services → Library → search "Google Sheets API" → Enable it.
4
Create OAuth credentials
APIs & Services → Credentials → Create Credentials → OAuth Client ID → Web Application.

Under Authorised JavaScript origins, add the URL you're hosting this file from. If using locally, add:
http://localhost
https://yourdomain.com
5
Configure OAuth consent screen
APIs & Services → OAuth consent screen → External → fill in App name (APEX Protocol), your email. Add scopes: ../auth/spreadsheets and ../auth/drive.file. Add your email as a test user.
6
Copy your Client ID and paste it below
Back on the Credentials page, copy your Client ID (ends in .apps.googleusercontent.com).
